# GatherGenius Data Permission Center

This package adds a user-controlled permission layer for personalization and voice intelligence.

Users can approve or deny:
- Profile + preferences
- Experience history
- Calendar availability
- Email context
- Photos/inspiration
- Location/region
- Payment preferences
- Provider network data
- Voice personalization

Permissions are stored locally in browser localStorage and optionally saved to Supabase table `user_data_permissions`.
