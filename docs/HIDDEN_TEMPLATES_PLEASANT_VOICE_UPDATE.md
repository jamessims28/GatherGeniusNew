# Hidden Templates + Pleasant Voice Update

## What changed

Visible template/tap grids were removed from the page.

GatherGenius now uses templates silently in the background:
- experience type templates
- vibe templates
- provider stack templates
- budget models
- approved user permissions
- voice/context personalization

## Voice update

The voice preview was changed to sound more calm, pleasant, and helpful.

Preview message:
“Hi, I’m GatherGenius. I’m here to make this easy. With your permission, I’ll use what I know to build the best experience for you, protect the outcome, and guide you calmly through each step.”

Speech settings:
- slower rate
- slightly warmer pitch
- reassuring tone

## User-facing flow

1. User types one sentence or uses voice.
2. GatherGenius applies templates in the background.
3. The system builds the Experience Lock.
4. User does not see or manage templates.
