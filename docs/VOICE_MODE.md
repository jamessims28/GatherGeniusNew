# GatherGenius Voice Mode

Added a dedicated Voice Mode.

## Files

- `lib/voice/voiceModeEngine.js`
- `components/VoiceModePanel.jsx`
- `app/voice-mode/page.js`

## Features

- Enable Voice Mode button
- microphone permission check
- browser speech recognition
- verbal Genius response
- transcript capture
- fallback messages if browser blocks voice

Use Chrome or Edge for best browser speech recognition support.
