# Background Templates Update

Visible template buttons were removed from the main page.

GatherGenius now references templates silently in the background:
- experience type templates
- vibe/style templates
- provider stack templates
- budget models
- accessibility preferences

User-facing flow:
1. User enters one sentence or uses voice.
2. GatherGenius detects the experience type and vibe.
3. Background templates enrich the build.
4. The Experience Lock is generated without showing template choices.
