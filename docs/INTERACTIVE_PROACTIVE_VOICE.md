# Interactive + Proactive Voice

Genius can now:
- respond when asked
- speak first when helpful
- ask for permission when background data would improve the result
- warn when a risk or issue appears
- offer pricing help when pricing data is found
- do soft idle check-ins

User control:
- Proactive Voice can be toggled on/off in the UI.
- Cooldown logic prevents constant speaking.
- Proactive moments are helpful and non-intrusive.
